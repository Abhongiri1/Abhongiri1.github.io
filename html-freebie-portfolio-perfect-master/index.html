Learning Outcome: Gained insights into how malware operates and developed skills in reverse engineering and malware analysis.

Setup & Configured a Firewall and Intrusion Detection System (IDS)


Objective: Implement and configure a firewall and IDS to protect a network.

Tools Needed: pfSense, Snort, Virtual Machine.

Description: Installed pfSense on a virtual machine to act as a firewall for my network. Configured the firewall rules to block unwanted traffic and allow legitimate traffic. Learning Outcome: Understand the role of firewalls and IDS in network security and gained hands-on experience configuring and managing them.
